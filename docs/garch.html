SVR-GARCH and ANN-GARCH Analysis

GARCH

Generalized Autoregressive Conditional Heteroskedasticity or GARCH is used to model and predict volitlity in financial time series data. The formula is as follows:

σ2t = ω + αε2t-1 + βσ2t-1

σ2t represents the volitlity, ε2t-1 is the square of returns from the previous observation, and σ2t-1 is the volitlity of the previous observation. ω, α, and β are all parameters calculated by looking at previous observations. After seeing the accuracy of volatility predictions in other papers using SVR-GARCH and ANN-GARCH [add citation], we based our implementations of SVR-GARCH and ANN-GARCH after those in Machine Learning for Financial Risk Management by Abdullah Karasan. For the time period of volitility, we looked at the 5 day, 21 day, and 63 day volitility, which in terms of stocks are a week, one month, and three months.

SVR-GARCH

SVR-GARCH is using Support Vector Regression in order to solve for the volitlity. With SVR we use the same variables of the previous observations in order to solve for the current, but allow SVR to solve for the equation that will predict the next observation. We only had to find the best hyperparameters for our model by using GridSearch.

ANN-GARCH

ANN-GARCH is very similar to SVR-GARCH in that we give the model the same variables, but allow for it to solve for the equation and create predictions. We again tuned the hyperparameters using GridSearch to find what is the best fit model.

SVR vs ANN

To test which model was more accurate we modeled on the first 80% of observations, and then tested the predictions against the last 20% of the data. We then took the Root Mean Square Error (RMSE) between the obsevations and predictions for each model, so the lower the error rate the more accurate the model. The models for SVR-GARCH proved to be more accurate than those of ANN-GARCH.
