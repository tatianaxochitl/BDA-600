Correlation Findings

For correlation we decided to look at the features of opening price, closing price, open/close difference, high, low, log returns, volatility, and volume traded to see if there is a commonality between the financial assets in terms of any of these. To analyze this we used linear regression for each pair and created a heatmap based on p value between the two assets being compared. The color scale for the p value is set for the mid point at .05 so those that fall below can be show to have a statistically significant correlation. We also created graphs in order to compare the trends of the different stocks to better help look at the correlation show by the p-values.
